macOS · Menu Bar

Sprout Pomodoro

A minimal menu bar Pomodoro timer with no Dock icon or window. Focus and break sessions auto-advance, with configurable durations (5–60 min) and system sound notifications when each session ends.

CLI · Python

PR Agent

A CLI tool that analyzes your git diff and generates intelligent PR descriptions. Supports Gemini via Vertex AI (default) and Claude Haiku 4.5 via GitHub Copilot, with a feedback loop to refine descriptions before creation.

macOS · Menu Bar

GCloud Session Watch

Shows a live H:MM:SS countdown until your Google Cloud Application Default Credentials expire. The menu bar icon turns amber under 10 minutes and red when expired — no network calls, no dependencies.
